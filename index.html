Click and drag to create elements. Go ahead, try it.

Type different letters to use different elements.

Hold down shift to make a "generator" that constantly spawns new elements.

Use the number keys to change the size of the "pen". 0 resets to default.

Use the arrow keys to move the little red guy and press space to shoot.

Bullets are specialized to explode with elements
